### Instructions to run
**SSH**
```shell
git clone git@github.com:Yoru-cyber/bookstore.git
```
**HTTPS**
```shell
git https://github.com/Yoru-cyber/bookstore.git
```
**Run build of gradle command**
```shell
./gradlew build
```
**Build and run the container**
```shell
Docker compose up --build
```
